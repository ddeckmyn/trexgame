# T-Rex Kopbal

Deze game bestaat uit statische bestanden (`index.html`, `style.css`, `script.js`).

## Lokaal draaien

1. Clone de repository naar je eigen machine en ga naar de map:
   ```bash
   git clone <jouw-repo-url>
   cd trexgame
   ```
2. Start een lokale server:
   ```bash
   python -m http.server 8000
   ```
3. Open de game in je browser:
   ```text
   http://localhost:8000
   ```
